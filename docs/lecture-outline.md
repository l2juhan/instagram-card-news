# 바이브코딩으로 인스타 카드뉴스 자동화 만들기 — 2시간 강의 구성안

> **대상**: 비개발자 | **방식**: 바이브코딩 (AI한테 시키는 코딩) | **소요시간**: 120분

---

## Track 1: 오프닝 & 왜 만드는가 (10분)

**핵심 메시지**: "코딩 몰라도 AI한테 시키면 됩니다"

- 완성된 카드뉴스 데모 시연 (한 줄 입력 → PNG 9장 나오는 장면)
- 기존 방식 vs 이 방식 비교
  - 캔바/미리캔버스: 매번 수작업, 디자인 고민
  - 이 프로젝트: 주제만 던지면 리서치 → 글쓰기 → 디자인 자동
- 오늘 만들 것의 전체 그림 (간단한 흐름도)
- 준비물 안내: Mac/PC, Claude Code 구독

---

## Track 2: 작업 환경 세팅 (15분)

**핵심 메시지**: "딱 3가지만 설치하면 끝"

| 순서 | 할 일 | 설명 |
|------|-------|------|
| 1 | Node.js 설치 | "프로그램을 실행시켜주는 엔진" — 공식 사이트에서 다운로드 클릭 |
| 2 | Claude Code 설치 | 터미널에 한 줄 복붙 |
| 3 | 프로젝트 폴더 만들기 | `mkdir instagram-card-news` |

- 터미널이 뭔지 30초 설명 ("컴퓨터한테 글로 명령하는 창")
- 각 설치를 화면 공유하며 같이 따라하기
- Claude Code 처음 실행 & 인증까지

---

## Track 3: AI한테 프로젝트 설계 시키기 (15분)

**핵심 메시지**: "내가 설계하는 게 아니라, AI한테 설계를 '주문'합니다"

- CLAUDE.md 파일의 역할 설명 ("AI한테 주는 업무 매뉴얼")
- 실습: 첫 번째 대화
  - "인스타그램 카드뉴스를 자동으로 만드는 프로젝트를 만들어줘"
  - AI가 폴더 구조, 필요한 파일 목록을 제안하는 과정 보여주기
- 바이브코딩의 핵심 원칙 3가지
  1. **구체적으로 주문하기** — "예쁘게" 대신 "1080x1350 사이즈, 밝은 배경"
  2. **한 번에 하나씩** — 한꺼번에 다 시키지 않기
  3. **결과 확인 후 다음 단계** — 중간중간 눈으로 확인

---

## Track 4: 디자인 템플릿 만들기 (25분)

**핵심 메시지**: "HTML은 몰라도 됩니다. AI한테 '이렇게 생긴 카드 만들어줘'라고 말하면 됩니다"

### 4-1. 첫 번째 템플릿 — 표지 (10분)

- AI한테 요청: "카드뉴스 표지 템플릿을 HTML로 만들어줘. 1080x1350, 제목이 크게, 부제가 작게"
- 만들어진 결과를 브라우저에서 열어보기
- 색상/폰트 수정 요청: "배경을 흰색으로, 폰트를 Pretendard로 바꿔줘"

### 4-2. 본문 템플릿 (10분)

- 다양한 슬라이드 타입 중 2~3개 핵심만 같이 만들기
  - `content` (기본 내용)
  - `content-stat` (숫자 강조)
  - `cta` (마지막 장)
- 나머지 타입은 같은 방식으로 추가하면 된다는 것만 안내

### 4-3. 스타일 변형 (5분)

- "이 템플릿을 어두운 버전으로도 만들어줘" → bold 스타일 탄생
- 템플릿 = 틀, 스타일 = 분위기라는 개념 설명

## Track 5: 자동 렌더링 기능 만들기 (15분)

**핵심 메시지**: "HTML 파일을 이미지로 바꿔주는 프로그램을 AI한테 만들어달라고 합니다"

- AI한테 요청: "slides.json 파일을 읽어서 각 슬라이드를 PNG로 저장하는 스크립트 만들어줘"
- Puppeteer 설명 (1문장): "보이지 않는 브라우저가 HTML을 열고 스크린샷을 찍어주는 도구"
- 실습: 테스트 데이터로 실행 → PNG 파일 확인
- 안 될 때 대처법: 에러 메시지를 그대로 AI한테 붙여넣기

---

## Track 6: AI 리서치 & 글쓰기 자동화 (20분)

**핵심 메시지**: "주제만 주면 AI가 알아서 조사하고, 카드뉴스 글을 써줍니다"

### 6-1. 리서치 자동화 (8분)

- CLAUDE.md에 리서치 지침 추가하기
- "이 주제에 대해 웹에서 검색해서 핵심 포인트 정리해줘" → research.md 생성
- research.md 결과물 같이 확인

### 6-2. 카피라이팅 자동화 (12분)

- CLAUDE.md에 카피라이팅 가이드라인 추가
- slides.json 포맷 설명 (어떤 모양의 데이터가 필요한지만)
- 실습: 리서치 결과 → slides.json 자동 생성 → 렌더링 → PNG 확인
- 이 과정이 한 줄 명령어로 전부 실행되는 것을 보여주기

## Track 7: 전체 파이프라인 연결 & 실전 시연 (15분)

**핵심 메시지**: "이제 한 줄이면 카드뉴스가 완성됩니다"

- 실전 라이브 시연 (주제 2개)
  - 예시 1: "AI 트렌드 2026 카드뉴스 만들어줘"
  - 예시 2: "건강한 아침 루틴 카드뉴스, bold 스타일로"
- 각 단계가 자동으로 진행되는 과정을 실시간으로 보여주기
- 완성된 카드뉴스를 인스타그램에 실제로 올리는 장면까지

---

## Track 8: 클로징 — 내 것으로 만들기 (5분)

- 커스터마이징 팁
  - 새 스타일 추가: "magazine 스타일 만들어줘, 이런 느낌으로"
  - 계정명/색상 변경: config.json 한 줄 수정
  - 슬라이드 타입 추가: "before-after 비교 슬라이드 타입 추가해줘"
- 바이브코딩 핵심 요약
  - 원하는 것을 구체적으로 말하기
  - 한 번에 하나씩 만들기
  - 안 되면 에러를 그대로 보여주기
- Q&A 안내

## 강의 설계 포인트

- **Track 4가 가장 길고 핵심** — 여기서 수강생이 "AI한테 시키는 감각"을 체득
- **매 Track 끝에 눈에 보이는 결과물** — 동기부여 유지
- **에러 대처를 자연스럽게 포함** — 실제 라이브에서 에러가 나면 "이렇게 대처합니다"로 활용
- **코드는 절대 설명하지 않음** — AI가 만든 코드를 열어보되 "이건 AI가 알아서 한 부분"으로 넘기기
